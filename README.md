## GitHub Pages Deployment

This website is automatically deployed to GitHub Pages at: **https://UCUmovenglab.github.io/ucumovenglab/**

The site is built and deployed automatically using GitHub Actions whenever changes are pushed to the `main` branch.

## Local Development

To run the site locally:

1. Install Ruby and Bundler
2. Run `bundle install` to install dependencies
3. Run `bundle exec jekyll serve` to start the development server
4. Visit `http://localhost:4000/ucumovenglab/` in your browser
